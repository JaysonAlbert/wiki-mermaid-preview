# Privacy Policy for Wiki Mermaid Preview

Last updated: 2026-04-01

## Overview

Wiki Mermaid Preview is a Chrome extension that renders Mermaid diagram previews directly below Mermaid source blocks on user-authorized wiki and documentation pages.

This extension is designed to work locally in the browser. It does not operate a backend service for collecting your Mermaid content or page data.

## What information the extension handles

The extension may handle the following information while it is running:

- Website content needed to detect Mermaid source blocks and render previews on pages you authorize
- Extension settings that you configure, including:
  - authorized site match patterns
  - Mermaid selector rules

## How the information is used

The extension uses this information only to provide its core functionality:

- detect Mermaid source blocks on pages you authorize
- render Mermaid previews below the source blocks
- remember your authorized sites and selector rules across browser sessions

## Storage

The extension stores its configuration using Chrome extension storage, including Chrome sync storage when available.

The stored configuration may include:

- authorized site match patterns
- selector rules and related rendering settings

This stored information is used only so the extension can continue working with your chosen sites and page structures.

## What the extension does not do

Wiki Mermaid Preview does not:

- sell your data
- use your data for advertising
- use your data for profiling unrelated to the extension's functionality
- use your data to determine creditworthiness or lending eligibility
- load or execute remotely hosted code
- send Mermaid page content or extension settings to the developer's own servers

## Data sharing

The extension does not share your Mermaid source content, page content, or extension settings with the developer or third parties for independent use.

## Security

All executable logic used by the extension is packaged with the extension itself. The extension does not load remotely hosted JavaScript or other remote executable logic.

## Changes to this policy

If the extension's data practices change, this privacy policy will be updated before or at the time those changes are released.

## Contact

For privacy questions or support requests, use the support contact listed on the Chrome Web Store page for this extension.
